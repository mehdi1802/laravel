## About This Project

This is a interview question project based on PHP MVC Framework (Laravel 5.7) for MFour Mobile Market Research.

This API has 4 routes:
- /login
- /users -> to retrieve all users
- /users/create/ -> to create a new user
- /users/update/{user_id} -> to update the existing user
